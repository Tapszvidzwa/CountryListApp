# CountryListApp
Android Mobile app that fetches list of countries and displays them in recyclerview

Tech stack used:
- Coroutines
- Kotlin
- Retrofit, OkHttp
- MVVM
- Data binding
- View binding
- Room Database
- Defensive programming techniques
- Error handling
